# HTTP Auth Helpers Example

Single-concept example: use secret() and auth helpers to build authorization headers.

How to use it:
- copy this folder to a new project directory
- set `N3_SECRET_EXAMPLE_KEY` or add `{ "example_key": "..." }` to `.namel3ss/secrets.json`
- run `n3 app.ai check`
- run `n3 run` and click "Fetch with auth" in the UI
