# First 5 minutes

0. If anything fails to run, start with `n3 doctor` for setup checks.
1. Scaffold the demo and run it
   - `n3 new demo demo`
   - `cd demo`
   - `n3 run`
2. Ask a question
   - Use the chat composer on the home page.
3. Review the answer and why fields
   - Open the Answer table to see reply + why.
4. Explore
   - Open **State** to see stored values.
   - Open **Traces** after a run to inspect ai inputs/outputs.
