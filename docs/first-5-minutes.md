# First 5 minutes

0. If anything fails to run, start with `n3 doctor` for setup checks.
1. Scaffold Operations Dashboard and run it
   - `n3 new operations_dashboard ops_app`
   - `cd ops_app`
   - `n3 run`
2. Seed the queue
   - Select "Create sample incident" to add a row.
3. Review the incident list
   - Scan the incident list and escalation checklist.
4. Explore
   - Open **State** to see stored values.
   - Open **Traces** after a run to inspect flow inputs/outputs.
