# First 5 minutes

1. Run a demo in Studio
   - `n3 examples/demo_product_dashboard.ai studio`
   - Open the URL shown and select the "home" page.
2. Edit a title inline
   - Click a title in the preview, edit the text, press Enter.
3. Add a card
   - Click the "Add element" dropdown, choose "card", click Add. A new card appears in the selected container.
4. Run a button action
   - Click the "Seed data" or "Refresh" button; observe the table update.
5. See state and traces
   - Open the State tab to see stored data.
   - Open the Traces tab after actions or AI calls to view inputs/outputs.
6. Explore
   - Switch pages via the selector, inspect elements in the Inspector, and move elements up/down to understand layout.
