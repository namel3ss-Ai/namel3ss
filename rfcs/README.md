# namel3ss RFCs

This folder tracks language and semantic proposals.

## Status folders
- `accepted/`
- `rejected/` (optional)
- `drafts/` (optional)

## Required fields for each RFC
- RFC ID
- title
- status
- motivation
- proposal
- alternatives
- compatibility and migration
- determinism impact
- test plan

See `TEMPLATE.md` for the standard structure.
